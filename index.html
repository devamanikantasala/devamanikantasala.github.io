Training the Chatbot

Training the chatbot was a critical part of the project. I utilized several predefined conversation sets to train the chatbot using ChatterBot's ListTrainer. This step ensured that the chatbot could handle basic conversational queries and respond appropriately. Additionally, I exported and cleaned a real-world WhatsApp chat to provide a rich dataset for training, enhancing the chatbot's conversational abilities.

Integrating Gemini AI

To improve the chatbot's responses when confidence levels were low, I integrated Gemini AI using its API key. This integration allowed the chatbot to leverage advanced generative AI capabilities for more accurate and contextually relevant responses. The integration process involved configuring the API, setting a confidence threshold, and using Gemini AI for fallback responses.
